TP2 - Service Web SOAP Calculatrice

- Implémentation service web SOAP avec Jakarta XML Web Services
- Interface ICalculatrice avec annotations @WebService
- Classe Calculatrice avec 4 opérations (somme, multiplication, soustraction, division)
- ServeurJWS pour publication avec Endpoint.publish()
- Dépendances Maven : jakarta.xml.ws-api, jaxws-rt, jakarta.jws-api
